# Algorithm_Study_2021

+ 목적

Coding Test 준비로 기본적인 프로그래머스 Lv1, Lv2 문제를 모두 푼 뒤 완전탐색을 기반으로 기본적인 코테 준비

+ 사용언어

Python 3

+ 스터디 계획

매주 토요일 오전 10시, 디스코드로 서로의 코드 확인하고 발표하기 + 첨삭

+ 스터디 시간

매주 토요일 오전 10시(약 두시간 동안 진행), 이후 일~금 스터디 준비: 개개인 문제 풀기

+ 스터디 기간

4~5개월 예상

+ 문제 계획
  + 프로그래머스 Lv1, Lv2 문제 (기초 위함)
  + 백준 완전 탐색(DFS, BFS)
  + Dynamic Programming
